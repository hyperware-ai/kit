# UqDev

Tools for developing on Uqbar

## Installing

Clone this repo, and then install with cargo:

```
git clone https://github.com/uqbar-dao/uqdev
cd uqdev
cargo install --path .
```

## Usage

```
uqdev --help
uqdev build --help
uqdev inject-message --help
uqdev start-package --help
uqdev run-tests --help
```
